# scaling-fojing

把现代汉语翻译成唐译梵文.

Translate modern Chinese into TSC (Tang-Sanskrit-Chinese).

##  安装和运行测试

```
git clone https://github.com/CheshireRabbit/scaling-fojing.git
cd scaling-fojing
pip install pypinyin
python han_to_tang.py 斯里兰卡
```
